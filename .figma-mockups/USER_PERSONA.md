# SkinTrack 目标用户画像

## 核心用户 — "护肤成分党" Lisa
- **年龄**: 22-32 岁女性
- **职业**: 白领/学生/自由职业
- **平台**: iPhone 为主，关注小红书/抖音美妆内容
- **消费力**: 愿意为有效的护肤品投资（月均护肤消费 300-800 元）
- **痛点**:
  - 用了很多产品但不知道哪个真正有效
  - 皮肤状态波动不知道原因（换季/压力/产品）
  - 想科学护肤但缺乏量化数据
  - 想记录变化但懒得写日记

## 次要用户 — "护肤新手" Mia
- **年龄**: 18-24 岁
- **场景**: 刚开始关注护肤，想建立科学的护肤习惯
- **需求**: 简单易用、有引导、不需要专业知识

## 设计调性关键词
**温暖 · 治愈 · 清新 · 科学感 · 有温度的数据**

### 视觉方向
- 色彩: 薄荷绿(科学) + 玫瑰粉(温暖) + 薰衣草紫(治愈) 三色穿插
- 语气: 鼓励式（"皮肤正在变好哦"）而非冰冷报告式
- 插画: 柔和线条 + 护肤品/植物元素
- 照片占位: 女性面部轮廓（柔和、优雅）
- 留白: 充足呼吸感，不拥挤
- 圆角: 大圆角为主，柔和感
