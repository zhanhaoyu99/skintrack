## 组件详细规格

### 1. 前后对比卡片 CompareCard
- **背景**: 三色渐变 `linear-gradient(135deg, #FFF8F2, #F0FAF6, #F5F0FF)` + 微投影
- **标题行**: "前后对比" + "📤分享" (primary 色链接) → ShareCardScreen
- **照片区域**: 两张照片并排，160dp 高，中间 VS 圆形徽章 (28dp)
- **照片样式**: 圆角 lg，底部渐变遮罩，左下分数标签 (13sp Bold 白色)，右下日期 (11sp)
- **评分变化**: 居中，大号分数差 (22sp Bold success 色) + 箭头图标
- **鼓励文案**: "皮肤在变好哦，继续加油~" (13sp onSurfaceVariant)

### 2. 趋势图卡片
- **标题行**: "趋势变化" (SectionHeader)
- **指标 Chips**: 横向滚动，选中态 primary 背景白字，未选中 surfaceVariant
  - 选项: 总评分 / 痘痘 / 毛孔 / 均匀度 / 水润
- **折线图**: Canvas 绘制，面积填充渐变，数据点圆标记
- **当前值**: 最右点白底+primary描边+光晕
- **X 轴**: 日期标签

### 3. 记录列表
- **标题**: "所有记录" (SectionHeader)
- **卡片样式**: 水平布局，surface 背景圆角 lg，微投影
  - **缩略图**: 56dp 方形圆角，皮肤色渐变背景 + 半透明人脸轮廓
  - **信息区**:
    - 第一行: 日期 (15sp Bold) + 变化标签 (11sp Bold, 圆角背景)
      - 上升: success 色 "↑+N" + #ECFDF5 背景
      - 持平: onSurfaceVariant "→ 0" + surfaceVariant 背景
      - 下降: error 色 "↓-N" + error 淡背景
    - 第二行: AI 摘要 (13sp onSurfaceVariant)
  - **迷你 ScoreRing**: 44dp，显示总分

- **点击**: → RecordDetailScreen
- **分页加载**: LazyColumn + 滚动触底加载更多
