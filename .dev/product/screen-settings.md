## 菜单分组

### 账户组
| 菜单项 | 图标 | 副标题 | 交互 |
|--------|------|--------|------|
| 编辑资料 | 人像 (Mint) | 头像、昵称、肤质 | → 编辑资料页 |
| 修改密码 | 锁 (Lavender) | 上次修改：30 天前 | → 修改密码页 |
| 会员管理 | 星星 (Apricot) | Pro 会员 · 2026.12.14 到期 | → PaywallScreen |

### 通知组
| 菜单项 | 图标 | 副标题 | 交互 |
|--------|------|--------|------|
| 打卡提醒 | 铃铛 (Rose) | 每天 20:00 | Switch 开关 |
| 周报通知 | 图表 (Lavender) | 每周日发送肌肤周报 | Switch 开关 |
| AI 分析通知 | 脑图 (Mint) | 分析完成后推送提醒 | Switch 开关 |

### 数据组
| 菜单项 | 图标 | 副标题 | 交互 |
|--------|------|--------|------|
| 数据同步 | 云 (Mint) | 最后同步：今天 09:30 + "已同步" badge | 点击手动同步 |
| 清除缓存 | 垃圾桶 (Grey) | 当前缓存 23.4 MB | 点击确认清除 |
| 导出数据 | 下载 (Info) | 导出为 JSON / CSV | → 导出弹窗 |

### 关于组
| 菜单项 | 图标 | 副标题 | 交互 |
|--------|------|--------|------|
| 版本信息 | 信息 (Grey) | v1.0.0 (Build 42) | 无 |
| 隐私政策 | 锁 (Grey) | 打开隐私政策 | → WebView |
| 用户协议 | 文档 (Grey) | 打开用户协议 | → WebView |

### 底部操作
- **退出登录**: 全宽 outlined 按钮 (error 色描边)
- **注销账户**: 文字链接 (error 色, 12sp), 位于退出按钮下方

### 分组标题样式
- 分组标签左侧有 3px primary 色竖条装饰 (`::before` 伪元素)

## 注销账户弹窗 (DeleteAccountDialog)

```
┌─────────────────────────┐
│                         │ ← 半透明遮罩
│  ┌────────────────────┐ │
│  │ ⚠️ 注销账户         │ │
│  │                    │ │
│  │ 此操作不可恢复，你的│ │
│  │ 所有数据（包括肌肤  │ │
│  │ 记录、分析报告、护  │ │
│  │ 肤品信息）将被永久  │ │
│  │ 删除。              │ │
│  │                    │ │
│  │ 请输入密码以确认     │ │
│  │ [输入你的密码    ]  │ │
│  │                    │ │
│  │  取消    [确认注销] │ │ ← 取消(灰) + 确认注销(红)
│  └────────────────────┘ │
└─────────────────────────┘
```

### 注销弹窗规格
- **标题**: "⚠️ 注销账户" (warning 图标 + 红色)
- **说明文案**: 不可恢复警告 + 数据删除范围说明
- **密码确认**: 输入框 + placeholder "输入你的密码"
- **按钮**: "取消" (灰色文字) + "确认注销" (红色背景按钮)
- **接口**: DELETE /api/auth/user (GDPR 合规)

## 编辑资料页 (子页面, V1 简化实现)

```
┌─────────────────────────┐
│ ← 编辑资料       [保存]  │
├─────────────────────────┤
│      [头像]              │ ← 点击更换头像 (V1: 仅首字母+渐变色)
│                         │
│  昵称                    │
│  [当前昵称            ]  │
│  邮箱                    │
│  [当前邮箱 (不可编辑)]   │ ← 灰色, disabled
└─────────────────────────┘
```

- **V1 范围**: 仅支持修改昵称, 头像为自动生成 (首字母+渐变)
- **V2 规划**: 支持头像上传, 性别/生日等

## 修改密码页 (子页面)

```
┌─────────────────────────┐
│ ← 修改密码               │
├─────────────────────────┤
│  当前密码                 │
│  [输入当前密码        👁] │
│  新密码                   │
│  [输入新密码 (≥6位)   👁] │
│  确认新密码               │
│  [再次输入新密码      👁] │
│                         │
│  [    确认修改    ]      │ ← primary 按钮
└─────────────────────────┘
```

- **校验**: 当前密码正确 + 新密码 ≥6 位 + 两次一致
- **成功**: Toast "密码修改成功" + 返回设置页
- **失败**: 显示错误提示 (当前密码错误 / 新密码不符合要求)

## 周报通知说明

- **发送时间**: 每周日 10:00 (不可自定义, V1)
- **内容**: 本周皮肤评分变化 + 产品使用统计 + 下周建议
- **实现**: V1 为本地通知 (NotificationManager), V2 接入 FCM 服务端推送
- **开关默认**: 关闭
