# SkinTrack 产品概览

## 一句话定位

**每天拍一张脸，AI 追踪皮肤变化，告诉你护肤品到底有没有用。**

## 核心价值主张

- **量化护肤效果**：不再只能"感觉"护肤品有没有效，用数据看得见
- **长期追踪**：竞品只做"单次检测"，SkinTrack 做"长期追踪 + 护肤品归因分析"
- **科学决策**：AI 分析哪个产品让你变好/变差，科学护肤不踩坑

## 目标用户

### 核心用户 — "护肤成分党" Lisa
- **年龄**: 22-32 岁女性
- **消费力**: 月均护肤消费 300-800 元
- **痛点**: 用了很多产品但不知道哪个真正有效；皮肤状态波动不知道原因
- **平台**: iPhone 为主，关注小红书/抖音美妆

### 次要用户 — "护肤新手" Mia
- **年龄**: 18-24 岁
- **场景**: 刚开始关注护肤，想建立科学习惯
- **需求**: 简单易用、有引导

## 设计调性

**温暖 · 治愈 · 清新 · 有温度的数据**

- 语气：鼓励式（"皮肤状态不错哦~"）而非冰冷报告
- 色彩：薄荷绿(科学/信任) + 杏色(温暖/活力) + 玫瑰粉(女性化) + 薰衣草紫(治愈)
- 圆角：大圆角为主，柔和感
- 留白：充足呼吸感

## 用户配置维度

### 肤质类型 (Onboarding P4 选择)
油性 / 干性 / 混合 / 敏感 / 中性 — 影响 AI 分析的基准模型

### 肌肤目标 (Profile 设置)
祛痘 / 收毛孔 / 提亮 / 淡斑 / 抗衰 / 保湿 等 — 影响 AI 建议方向

## 核心用户旅程

```
首次使用:  引导页(4P) → 注册 → 首页(空状态) → 拍第一张自拍 → AI分析 → 记录详情
日常使用:  首页 → 拍照 → AI分析 → 记录产品 → 查看趋势
深度使用:  首页 → 归因报告 → 了解产品效果 → 分享对比卡片
付费转化:  14天免费 → 触发门控 → 查看权益 → 订阅
```

## 功能优先级

### P0 核心 (V1.0)
1. 每日拍照 AI 分析（痘痘/毛孔/均匀度/泛红/水润 五维评分）
2. 时间线趋势图 + 前后对比
3. 护肤品记录（AM/PM 分组 + 每日打卡）
4. 拍照标准化引导（人脸框 + 提示文字）

### P1 关键差异化 (V1.0)
5. 归因分析报告（AI 关联产品使用与皮肤变化）
6. 打卡连续 + 里程碑激励

### P2 增强 (V1.5-V2.0)
7. 饮食/睡眠/姨妈期记录
8. 护肤品效果排行榜（众包数据）
9. 社区功能（分享对比图、产品评价）

## 皮肤评分体系

### 五维指标（0-100 分）

| 指标 | 含义 | AI 分析维度 |
|------|------|------------|
| 痘痘 (acne) | 痘痘/痤疮数量和严重程度 | 数量、位置、炎症程度 |
| 毛孔 (pore) | 毛孔大小和可见度 | T区/脸颊毛孔、黑头 |
| 均匀度 (evenness) | 肤色均匀程度 | 色斑、暗沉、肤色不均 |
| 泛红 (redness) | 皮肤泛红程度 | 泛红区域、红血丝 |
| 水润 (hydration) | 皮肤水润度 | 干燥程度、光泽感 |

### 总评分

加权平均，四级档位：
- **EXCELLENT** (85-100): "皮肤状态很棒哦!"
- **GOOD** (70-84): "皮肤状态不错~"
- **MODERATE** (55-69): "还有提升空间"
- **CONCERN** (0-54): "需要更多关注"
