## 组件详细规格

### 1. TopAppBar
- 返回箭头 + "护肤品管理" 标题 + "+" 添加按钮
- 添加按钮 → 弹出 AddProductSheet (BottomSheet)

### 2. 搜索栏
- 全宽圆角搜索框
- placeholder: "搜索产品名称或品牌"
- 实时过滤产品列表

### 3. 分类筛选 Chips
- 横向滚动，选中态 primary 背景白字
- 选项: 全部 / 洁面 / 精华 / 面霜 / 防晒 / 化妆水 / 面膜 / 其他

### 4. 今日打卡进度
- 进度条: primary 色填充条
- 文字: "今日打卡进度" + "2/5" (右侧)
- 未记录提醒: Rose 色背景卡片，"⚠️ 今天还有 N 个护肤品没记录哦~"

### 5. AM/PM 分组
- **分组标题**: "☀️ AM 早间护肤" / "🌙 PM 晚间护肤" + 打卡统计 "2/3 已打卡"
- **右侧**: 打卡进度文字

### 6. 产品卡片
- **图标**: 产品品类图标，彩色圆角背景
- **信息**:
  - 产品名 (15sp Bold)
  - 品牌名 · 英文名 (13sp onSurfaceVariant)
  - 品类标签 (小 pill，category 特定色)
- **打卡按钮**: 右侧
  - 已打卡: primary 绿色实心圆 + 白色勾 (表示完成)
  - 未打卡: 空心圆 (outlineVariant 描边)
- **点击卡片**: 查看产品详情/编辑
- **长按**: 删除产品
- **滑动删除**: 右滑显示删除

## AddProductSheet (BottomSheet)
- 产品名输入
- 品牌输入
- 品类选择 (Dropdown)
- 使用时段 (AM/PM/Both)
- 保存按钮
