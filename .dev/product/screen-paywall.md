## 组件详细规格

### 1. 关闭按钮
- 左上角 "✕" 图标，灰色
- 点击返回上一页

### 2. 品牌区域
- **背景**: 顶部渐变 `linear-gradient(180deg, rose-50, lavender-50, transparent)`
- **皇冠图标**: 80dp 圆形，金色渐变 `linear-gradient(145deg, #FFD700, #FFAA00, #FF8C00)` + 光晕阴影，白色皇冠 SVG
- **标题**: "解锁你的变美潜力" (24sp Bold)
- **副标题**: "科学护肤，让每一分投入都看得见效果" (15sp onSurfaceVariant)

### 3. 试用提示
- primary 色文字 + ⓘ 图标
- "新用户享 14 天免费试用"
- 仅未订阅且在试用期内显示

### 4. 权益列表
- 5 条权益，每条前置 ✅ 勾选图标
- 16sp 正文，行间距适中

| # | 权益描述 |
|---|---------|
| 1 | 无限拍照记录，不受次数限制 |
| 2 | AI 详细分析报告与个性化建议 |

#### 月度会员
- **标题**: "月度会员"

#### 年度会员（推荐）
- **标题**: "年度会员"

- **默认选中**: 年度会员

### 6. 社交证明 + 信任信号（新增）
- **社交证明**: "12,580 位用户已订阅" + 3 个头像圆形重叠组
- **信任信号**: 3 项水平排列 (12sp onSurfaceVariant)
  - "🔒 安全支付" / "⟳ 随时取消" / "⚡ 即时生效"
- **恢复购买**: "恢复购买" 文字链接 (primary 色, 底部)

### 7. 订阅按钮
- 全宽 primary 按钮
- 文案: "立即订阅"
- 加载态: 显示 spinner
- 成功: 关闭页面 + Toast "订阅成功"

## 支付流程
```
选择方案 → 点击订阅 → PaymentManager.purchase()
  → 成功: 更新 SubscriptionRepository → dismiss → Toast "订阅成功"
  → 支付取消: 恢复按钮状态，不显示错误
  → 网络错误: 显示 "网络连接失败，请检查后重试" + 重试按钮
  → 支付失败: 显示 "支付失败: {原因}" + 重试按钮
```
