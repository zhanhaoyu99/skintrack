## 组件详细规格

### 1. 照片区域
- **高度**: 280dp
- **背景**: 肤色渐变 `linear-gradient(170deg, #D4B8A8, #C8A898, #B8A088, #A89480)`
- **人脸占位**: 半透明 SVG 椭圆轮廓 (opacity 0.08)
- **TopBar**: 悬浮在照片上方，返回按钮 + 分享按钮（圆形半透明黑背景）
- **评分标签**: 底部居中悬浮，模糊背景 (`rgba(0,0,0,0.4)` + `backdrop-filter: blur(12px)`)
  - 内容: ScoreRing 迷你图标 + "82 分 · 良好"

### 2. 悬浮评分卡片
- **定位**: margin-top -40dp，浮在照片区域上
- **背景**: surface + shadow-lg
- **圆角**: xl
- **内容**: ScoreRing (74dp) + 标题 "肌肤状态良好" (19sp Bold) + 日期 (13sp) + 变化 pill
- **变化 pill**: success 背景 (#ECFDF5) + "↑ +4 较上次" + 箭头图标

### 3. 雷达图 RadarChart
- **卡片标题**: "多维评分"
- **尺寸**: 190dp x 190dp
- **6 个维度** (顺时针从 12 点方向): 整体 → 毛孔 → 泛红 → 水润 → 均匀 → 痘痘
- **样式**: 3 层六边形同心网格 (outlineVariant 0.5px) + 6 条轴线 (outlineVariant 0.5px)
- **数据面**: primary fill opacity 0.08 + primary stroke 2px + 顶点圆标记 4px (primary fill)
- **标签**: 各维度名称在六边形外侧 (11sp, onSurfaceVariant, font-weight 600)

### 4. 指标条形图
- **卡片标题**: "各项指标"
- **每行**: 指标名 (52dp 宽) + 渐变条形图 (7dp 高, 圆角) + 分值 (15sp Bold) + 变化值

| 指标 | 条渐变 (左→右) | 变化色规则 |
|------|----------------|-----------|
| 痘痘 | `#FF8A8A` → `#FF6B6B` | ≥3: success(绿) / 1-2: onSurfaceVariant / ≤-1: error(红) |
| 毛孔 | `#7EDCD4` → `#4ECDC4` | 同上 |
| 均匀度 | `#FFEE8D` → `#FFE66D` | 同上 |
| 泛红 | `#F5A1A1` → `#F38181` | 同上 |
| 水润 | `#75C7E1` → `#45B7D1` | 同上 |

> 渐变方向: `linear-gradient(90deg, 浅色, 深色)`，深色为 `DESIGN_SYSTEM.md` 中的指标色。

### 5. AI 分析卡片
- **背景**: 渐变 `linear-gradient(135deg, #F0FAF6, #F5F0FF)` + primary 微边框
- **标签**: "✅ AI 智能生成" pill (primaryContainer 背景)
- **文案**: 详细分析文本 (14sp, 1.7 行高)
- **门控**: 非会员显示 LockedFeatureCard 替代

### 6. 当日使用产品
- **标题**: "当日使用产品"
- **样式**: Chip/Tag 横向流式排列 (FlowRow)
- **每个 Chip**: 圆角 pill, surfaceVariant 背景, 13sp, 图标 + 产品名

## 门控规格
