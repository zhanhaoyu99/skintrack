# 工作流：部署前检查

## 触发命令
`/app deploy` — 部署前全面检查

## 执行流程

### 第1步：加载知识
- `knowledge/devops-deploy.md` — 部署规范
- `knowledge/security-checklist.md` — 安全检查
- `app-progress.md` — 当前状态

### 第2步：构建检查
```
[ ] 后端编译通过 (./gradlew :server:build)
[ ] 客户端编译通过 (./gradlew :composeApp:build)
[ ] 所有测试通过
[ ] 无编译警告（或已确认可忽略）
```

### 第3步：安全检查
```
[ ] 无硬编码密钥/密码
[ ] JWT Secret 使用环境变量
[ ] 数据库连接使用环境变量
[ ] ddl-auto 设为 none (生产)
[ ] show-sql 设为 false (生产)
[ ] CORS 限制域名 (生产)
[ ] HTTPS 配置正确
```

### 第4步：配置检查
```
[ ] application-prod.yml 配置正确
[ ] 数据库迁移脚本完整
[ ] 环境变量文档已更新
[ ] 日志级别合理
```

### 第5步：客户端检查
```
[ ] Android签名配置
[ ] iOS证书/Profile
[ ] 版本号已更新
[ ] 图标/启动页正确
[ ] 网络权限声明
```

### 第6步：功能检查
```
[ ] 核心功能手动验证通过
[ ] API端点全部可访问
[ ] 错误处理正常
[ ] 离线/弱网情况处理
```

```
━━━ 配置 ━━━

━━━ 功能 ━━━

[可以部署 / 有X个问题需先解决]
```
