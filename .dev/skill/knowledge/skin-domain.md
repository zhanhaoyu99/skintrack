# 皮肤领域知识库

## 皮肤类型分类

| 类型 | 英文 | 特征 | 护理重点 |
|------|------|------|---------|
| 油性 | OILY | T区出油旺盛，毛孔粗大，易长痘 | 控油、清洁、收缩毛孔 |
| 干性 | DRY | 紧绷、脱皮、细纹明显 | 保湿、修复屏障 |
| 混合偏油 | COMBINATION_OILY | T区油、两颊偏干 | 分区护理、T区控油 |
| 混合偏干 | COMBINATION_DRY | T区微油、两颊干 | 分区护理、保湿为主 |
| 敏感 | SENSITIVE | 易红、刺痛、对刺激敏感 | 修复屏障、简化护肤 |
| 中性 | NORMAL | 水油平衡、毛孔细腻 | 维持平衡、防晒 |

### 严重度等级
```
MILD     — 轻微，不影响外观
MODERATE — 中度，可见但可控
SEVERE   — 严重，需要重点关注
```

## 核心护肤成分

### 保湿成分
| 成分 | 功效 | 适合肤质 | 注意事项 |
|------|------|---------|---------|
| 玻尿酸（透明质酸）| 强力保湿 | 全肤质 | 干燥环境可能反吸水 |
| 甘油 | 保湿吸湿 | 全肤质 | 浓度过高可能黏腻 |
| 神经酰胺 | 修复屏障 | 敏感/干性 | 无 |
| 角鲨烷 | 柔润保湿 | 干性/敏感 | 油皮慎用高浓度 |

### 功效成分
| 成分 | 功效 | 浓度参考 | 注意事项 |
|------|------|---------|---------|
| 烟酰胺（维B3）| 美白、控油、抗老 | 2-5% | 不耐受者从低浓度开始 |
| 维C（抗坏血酸）| 美白、抗氧化 | 10-20% | 不稳定，需避光 |
| 水杨酸（BHA）| 疏通毛孔、去黑头 | 0.5-2% | 孕妇禁用 |
| 果酸（AHA）| 去角质、提亮 | 5-10%（家用）| 需配合防晒 |
| 维A醇（视黄醇）| 抗老、淡纹、控油 | 0.1-1% | 需建立耐受，孕妇禁用 |
| 壬二酸 | 祛痘、淡斑 | 10-20% | 相对温和 |
| 积雪草提取物 | 修复、舒缓 | — | 敏感肌友好 |
| 茶树精油 | 抗菌、祛痘 | 点涂 | 不可大面积使用 |

## 成分搭配禁忌

### 🔴 不建议同时使用
| 成分A | 成分B | 原因 |
|-------|-------|------|
| 维A醇 | 高浓度酸类（AHA/BHA）| 双重刺激，屏障受损 |
| 维A醇 | 维C | 稳定性互相影响 |
| 维C | 烟酰胺（高浓度）| 可能产生烟酸，刺激 |
| 酸类 | 酸类（叠加）| 过度去角质 |
| 酒精 | 敏感肌任何活性 | 加重刺激 |

### 🟢 推荐搭配
| 成分A | 成分B | 效果 |
|-------|-------|------|
| 维C | 维E | 协同抗氧化 |
| 烟酰胺 | 透明质酸 | 美白+保湿 |
| 神经酰胺 | 角鲨烷 | 修复+保湿 |
| 水杨酸 | 烟酰胺 | 控油+收毛孔 |

## 皮肤状态评估维度

### 评分体系（App内使用）
| 维度 | 评估指标 | 分数范围 |
|------|---------|---------|
| 水油平衡 | 出油量、干燥度 | 0-100 |
| 肤色均匀 | 色斑、暗沉、红血丝 | 0-100 |
| 纹理 | 毛孔、粗糙度、细纹 | 0-100 |
| 光泽度 | 健康光泽感 | 0-100 |
| 问题严重度 | 痘痘、闭口等问题的严重程度 | 0-100（低=严重） |

### 综合评分
```
总分 = 水油平衡×0.25 + 肤色均匀×0.2 + 纹理×0.2 + 光泽度×0.15 + (100-问题严重度)×0.2
```
